# Medical Abstracts Corpus Builder

This project aims to build a structured corpus of **medical abstracts and keywords** collected from publicly available medical publication platforms.

The corpus will support research in areas such as:
- Natural Language Processing (NLP)
- Information Retrieval
- Knowledge Organization
- Medical Text Mining
- Large Language Models for biomedical literature

## Project Overview

The repository contains Python scripts designed to automatically collect and structure data from medical publication platforms.

The scraped data includes:
- Article titles
- Abstracts
- Keywords
- Publication metadata (authors, year, journal, etc.)

The resulting dataset will be organized into a clean and reusable **corpus for computational analysis**.

## Data Sources

Potential sources include publicly accessible platforms such as:

- PubMed
- Semantic Scholar
- Open-access medical journals
- Other biomedical literature repositories

Only **publicly available metadata and abstracts** will be collected.

## Technologies Used

- Python
- BeautifulSoup
- Requests
- Selenium (if needed for dynamic pages)
- Pandas
- JSON / CSV for corpus storage
